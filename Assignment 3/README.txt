Summary of Program:

This program scans one or more text files and counts the number of occurences of each word in the file. The program uses 
a binary tree to keep track of all the words. It then prints all the words to an output file when all the files have been
scanned.

How to run:

To run this program invoke it from the command line as such:
	./PA3 outputFile inputFile1 inputFile2 ...
